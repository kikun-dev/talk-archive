# 坂道トーク帖 要件定義

## 概要

「坂道トーク帖」は、坂道メンバーから届いた言葉と時間を残す。私だけの記録帖。
外部アプリなどで受け取ったトークを保存・閲覧するための個人用Webアプリとする。

リアルタイムチャットではなく、**トーク履歴の保存と検索を目的とする。**

---

## 利用者

- 単一ユーザー
- 認証必須
- 個人利用のみ

---

## 会話メタデータ

- 各会話は **乃木坂 / 櫻坂 / 日向坂** のいずれか 1 グループに属する
- グループトークも必ずいずれか 1 グループに属する
- 各会話は相手側の参加者を 1 人以上持つ
  - 自分自身は参加者として保持しない
  - 1 人トークか複数人トークかは参加者数で判断する
  - participant は record の発言者参照に使うため、record 作成後の削除は許可しない
  - participant 編集は「名前変更」と「追加」を前提にし、既存 participant の ID を維持する
- 各会話はトップ画像を持てる
  - トップ画像は任意
  - 未設定時はプレースホルダー画像を表示する
- 各会話は会話期間を持つ
  - 会話期間は 1 つ以上の区間で管理する
  - 区間は開始日必須、終了日は任意
  - 終了日未入力は「現在進行中」を表す
  - 会話期間の日数は区間の和集合を日単位で数える
  - 区間の間にトークを受け取れないギャップがあれば、その日数は含めない
  - 現在進行中の区間は今日までの日数を表示する
  - 実際にトークが送られてきた日数とは別概念とする

## 保存するトーク形式

### テキスト

- タイトル（任意）
- テキスト（必須）
- 投稿日時（必須）
- 発言者（必須）
- 投稿日時は JST 基準で入力・表示する

### 画像

- タイトル（任意）
- テキスト（任意）
- 画像ファイル（1枚。未添付保存する場合を除き必須）
- 投稿日時（必須）
- 発言者（必須）
- 投稿日時は JST 基準で入力・表示する
- 詳細ページでインライン表示する

### 動画

- 動画ファイル（未添付保存する場合を除き必須）
- 音声あり / なし
- 投稿日時（必須）
- 発言者（必須）
- 投稿日時は JST 基準で入力・表示する
- 詳細ページで HTML5 プレーヤー再生できる

### 音声

- 音声ファイル（未添付保存する場合を除き必須）
- 投稿日時（必須）
- 発言者（必須）
- 投稿日時は JST 基準で入力・表示する
- 詳細ページで HTML5 プレーヤー再生できる

### メディア未添付レコード（#113）

- 画像 / 動画 / 音声は、composer で「ファイルはあとで添付する」を選んだ場合のみ
  メディアファイルなし（種別・投稿日時・発言者のみ）で保存できる
- 未添付かどうかは attachment の有無で判定する（状態カラムは持たない）
- 未添付レコードはタイムライン上で「◯◯未添付」バッジ表示され、その場でファイルを添付できる
- 添付ファイルの種別・サイズ制限は通常のメディアレコードと同じ（画像 10MB / 動画・音声 50MB）

## トーク詳細画面

- トーク詳細画面では、会話メタデータ本文ではなくトークのやり取りを主に表示する
- 上部に固定ヘッダーを持つ
  - トークタイトル
  - 会話内検索ボタン
  - 三点リーダ
- 三点リーダから以下へ遷移できる
  - 概要
  - 日付検索
  - 会話内メディア一覧
  - 会話編集（詳細画面内の編集モード切替）
  - 日付検索は詳細画面上のモーダルで開く
  - 会話メタデータ編集は概要画面から行う
- タイムラインは LINE 風の吹き出しレイアウトにする
  - participant 名とイニシャル丸アイコンを表示する
  - 各吹き出しに投稿日時を表示する
  - レコードの編集・削除は編集モード中のみ操作できる
- トーク追加は画面下部のタイプ切替式 composer で行う
  - 先にタイプを選び、そのタイプに必要な入力だけ表示する
  - participant が 1 人の会話では speaker 選択を省略して自動設定する
- 会話内検索は全文検索で、絞り込みではなく該当位置へジャンプする
- 日付検索はモーダルを閉じるだけで詳細画面へ戻れる
- 会話内メディア一覧からは詳細画面へ戻りやすい導線を持つ

---

## 主な機能

- 会話管理
  - グループ切替（乃木坂 / 櫻坂 / 日向坂）
  - カード形式のトーク一覧
  - カードにトップ画像、トーク名、会話期間日数を表示
- トーク保存
- メディア保存
- メディア表示
  - 画像のインライン表示
  - 動画の HTML5 プレーヤー表示
  - 音声の HTML5 プレーヤー表示
  - 非公開 Storage には signed URL でアクセスする
- 全文検索
- 会話内メディア一覧
  - 画像 / 動画 / 音声を会話ごとに分けて表示する
- トークJSON一括インポート（Phase 6 / #114）
  - インポート前にプレビュー画面で総件数・取り込み対象件数・重複スキップ件数・期間・種別内訳・行エラーを確認できる
  - 既存レコードとの重複（participant + 投稿日時 + 種別 + 本文先頭20文字）は自動でスキップする
  - メディア未添付のレコード（ファイルはあとから添付）としてまとめて取り込める
