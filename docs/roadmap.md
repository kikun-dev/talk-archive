## v2 以降（2次開発で検討する機能）

v1 完了後に優先度を判断して取り組む。

### データ強化

- **OCR インポート**: 画像からテキストを抽出して検索可能にする
- **一括インポート**: 外部アプリからのエクスポートデータを一括取り込み
- **バックアップ / エクスポート**: データの定期バックアップと外部エクスポート機能

### 検索強化

- **全文検索の高度化**: pg_trgm から外部検索エンジン（Meilisearch 等）への移行検討
- **ファジー検索 / 類似検索**: 曖昧な検索クエリへの対応
- **検索結果のソースフィルタリング**: 検索結果をソース別に絞り込み
- **検索履歴 / サジェスト**: よく使う検索クエリの保存と候補表示

### メディア強化

- **メディア圧縮**: アップロード時の自動圧縮・リサイズ
- **画像のライトボックス表示**: フルスクリーンでの画像閲覧
- **動画サムネイル自動生成**: 動画一覧でのプレビュー表示
- **複数画像の同時アップロード**: 1レコードに複数画像を添付

### UX 改善

- **ドラッグ & ドロップアップロード**: ファイルをドロップで追加
- **無限スクロール**: ページネーションの代替
- **レコードの並べ替え**: タイムライン内のレコード順序を手動変更
- **ダークモード**: テーマ切り替え対応
- **PWA 対応**: オフライン閲覧・ホーム画面追加

### システム

- **マルチユーザー対応**: 複数ユーザーでの利用（共有は不要でも個別アカウント）
- **API 提供**: 外部ツールからのデータ投入用 REST/GraphQL API
- **通知 / リマインダー**: 特定のトークに対するリマインダー機能
- **タグ / ラベル機能**: 会話やレコードへのタグ付けによる整理
